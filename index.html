    petstore-openapi-client

    A production-ready Petstore API built with FastAPI.

    This Python package is automatically generated by the OpenAPI Generator project:

    • API version: v1
    • Package version: 0.0.0
    • Generator version: 7.23.0-SNAPSHOT
    • Build package: org.openapitools.codegen.languages.PythonPydanticV1ClientCodegen

    Requirements.

    Python 3.7+

    Installation & Usage

    Install with uv

    Install runtime dependencies and the package in editable mode:

    uv venv
uv pip install -r requirements.txt
uv pip install -e .

    You can also install directly from Git:

    uv pip install git+https://github.com/RamonAlcantaraArceo/petstore-openapi-client-pydantic.git

    Then import the package: 

    import petstore_openapi_client

    Tests

    Execute tests with:

    uv pip install -r test-requirements.txt
uv run pytest

    Getting Started

    Please follow the installation procedure and then run the following:

    import asyncio 
import time
import petstore_openapi_client
from petstore_openapi_client.rest import ApiException
from pprint import pprint

# Defining the host is optional and defaults to http://localhost:8000
# See configuration.py for a list of all supported configuration parameters.
configuration = petstore_openapi_client.Configuration(
    host = "http://localhost:8000"
)


async def main():

    # Enter a context with an instance of the API client
    async with petstore_openapi_client.ApiClient(configuration) as api_client:
        # Create an instance of the API class
        api_instance = petstore_openapi_client.HealthApi(api_client)

        try:
            # Health Check
            api_response = await api_instance.health_check_api_v1_health_get()
            print("The response of HealthApi->health_check_api_v1_health_get:\n")
            pprint(api_response)
        except ApiException as e:
            print("Exception when calling HealthApi->health_check_api_v1_health_get: %s\n" % e)


if __name__ == '__main__':
    asyncio.run(main())
